Client examples — gọi gateway từ 9 ngôn ngữ

Gateway = JSON-over-WebSocket kiểu Binance tại ws://127.0.0.1:9001/. Cùng 1 socket cho cả 3 kiểu gọi.

3 kiểu gọi

KiểuGửiNhậnPhân biệt
① Stream (push){method:"SUBSCRIBE",params:["cme.nqu6@trade"],id}n lần {stream,data}theo stream
② RPC inline (query){method:"instrumentInfo",params:{…},id}1 lần {id,result} (lỗi: {id,error})theo id
③ RPC triggerSUBSCRIBE *.*@onpnlreplay → {method:"replayPnl",…,id}ack {id} + data về stream @OnPnlReplayack theo id, data theo stream
Cách phân loại message giống nhau ở mọi ngôn ngữ: có khoá stream → push; có khoá id → trả lời RPC. RPC inline trả result ngay; RPC trigger chỉ trả ack, data đến qua stream @On*. Stream key không phân biệt hoa thường. Mỗi ví dụ dưới làm cả 3 trong 1 client tối thiểu.
